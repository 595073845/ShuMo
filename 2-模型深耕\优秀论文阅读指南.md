# 优秀论文阅读指南

> **目标**: 从优秀论文中提取可复用的写作套路、模型选择思路、排版规范  
> **数量**: 阶段二精读至少10篇（近5年的获奖论文）  
> **本地已有**: CUMCM 92-93年论文（[[现有资料索引]] 中，年代较早，仅供参考结构）

---

## 优秀论文获取渠道

1. **CUMCM官网** (www.mcm.edu.cn) — 官方发布获奖论文
2. **数学建模微信公众号** — "数学建模学习交流"、"数模乐园"经常推送
3. **百度文库/道客巴巴** — 搜"202x年数学建模优秀论文"
4. **学长学姐** — 往届参赛者的论文最宝贵
5. **学校数模社团** — 一般有历年论文库

---

## 三遍阅读法

### 第一遍：快速浏览（15min/篇）
**目标**：判断这篇论文值不值得精读

看什么：
- [ ] 摘要 — 够不够清晰有力？
- [ ] 整体排版 — 是否规范美观？
- [ ] 模型复杂度 — 是简单模型+好解释，还是堆砌模型？
- [ ] 结果呈现方式 — 图表质量如何？

**筛选标准**：优先看 **国一论文**，省一论文可以参考但不必精读。

---

### 第二遍：结构拆解（30min/篇）
**目标**：学习论文的骨架

对照 [[论文写作指南]] 的标准结构，逐段分析：

```
1. 摘要 → 用了什么公式结构？怎么组织三个问题的结果？
2. 问题分析 → 做了什么样的图表呈现思路？
3. 模型假设 → 做了几条假设？为什么做这些假设？
4. 模型建立 → 公式推导到什么深度？模型之间如何衔接？
5. 求解方法 → 用了什么算法？是自己写还是调包？
6. 结果分析 → 用什么样的图表展示结果？
7. 模型检验 → 做了敏感性分析还是误差分析？
8. 优缺点 → 写了几条？是否客观？

→ 把每部分值得借鉴的点记到 Obsidian 里
```

---

### 第三遍：深度复盘（45min/篇）
**目标**：把论文"为我所用"

思考：
1. **如果我遇到这道题，我会怎么做？**（对比论文的做法）
2. **这个模型的选择好在哪里？**（有没有更好的选择？）
3. **摘要的这种写法有什么了不起？**（逐句分析）
4. **排版上有哪些我可以直接用的小技巧？**
5. **如果我是阅卷老师，这篇论文哪里打动我？**

---

## 阅读重点优先级

| 优先级 | 部分 | 原因 |
|--------|------|------|
| ⭐⭐⭐ | 摘要 | 最值得学，决定论文命运 |
| ⭐⭐⭐ | 模型选择逻辑 | 学会"为什么选这个模型" |
| ⭐⭐⭐ | 图表呈现 | 好图胜过千言万语 |
| ⭐⭐ | 问题分析思路 | 展示思考深度 |
| ⭐⭐ | 模型检验部分 | 容易被忽视，但加分项 |
| ⭐ | 附录代码 | 看风格，不用逐行理解 |

## 论文对比阅读

看完5篇后做一次横向对比：

| 维度 | 论文A | 论文B | 论文C |
|------|-------|-------|-------|
| 摘要清晰度 | 9/10 | 7/10 | 8/10 |
| 模型复杂度 | 中 | 高 | 中 |
| 图表质量 | 好 | 一般 | 好 |
| 可读性 | 高 | 中 | 高 |

从中找自己的定位：**你想要什么样的风格？**

---

## 推荐的阅读顺序

```
第1篇: 看"好论文长什么样"
第2-4篇: 同一年不同题（A/B/C各一篇），理解选题差异
第5-7篇: 你打算做的题型（如C题数据类）的近三年论文
第8-10篇: 同一种模型在不同论文中的应用
```
