## 🔷 建模手深入计划

### 核心教材
- 《数学建模算法与应用》（司守奎）— **重点啃完前8章**
- 《数学模型》（姜启源）— 入门阅读
- 配合 [[推荐网课]] 清风建模课

### 第3-4周学习清单
- [ ] 第1类：评价模型（AHP、TOPSIS、模糊评价、熵权法）— 三天拿下
- [ ] 第2类：预测模型（回归、灰色、时间序列、神经网络）— 三天
- [ ] 第3类：优化模型（线性、整数、非线性规划）— 两天
- [ ] 第4类：分类/聚类（决策树、SVM、K-means）— 两天
- [ ] 每天总结一个模型的：适用条件 + 输入输出 + 优缺点

### 第5周（交叉补位）
- [ ] 给编程手讲清楚你理解的模型数学原理
- [ ] 和写作手一起确定论文中模型描述部分的表达
- [ ] 准备自己的模型速查表（markdown格式，放进Obsidian）

### 快速阅读指南
不要求把书从头读到尾，遇到一个模型按以下流程：
```
1. 读模型简介（1min）
2. 看适用条件（2min）
3. 理解算法流程，跳过推导（5min）
4. 看例子（5min）
5. 找Python代码跑一遍（10min）
```

## 🔷 编程手深入计划

### 核心教材/资源
- **《数学建模算法与应用》配套代码**（GitHub搜"数学建模Python"）
- **Scikit-learn 官方文档**（分类/回归/聚类）
- **SciPy 官方文档**（优化/积分/插值）
- **清风课程代码部分**（[[推荐网课]]）

### 第3-4周学习清单
- [ ] 数据预处理模板（缺失值处理、异常值检测、标准化、PCA降维）
- [ ] 预测模型实现（回归、灰色预测、时间序列ARIMA、简单神经网络）
- [ ] 评价模型实现（AHP打分、TOPSIS、熵权法）
- [ ] 优化模型实现（scipy.optimize解线性/非线性规划）
- [ ] 分类/聚类实现（K-means、决策树、SVM、随机森林）
- [ ] 可视化美化模板（[[Python三件套速查]] 进阶+Plotly）
- [ ] 每种模型保存一个 `.py` 模板文件

### 第5周（交叉补位）
- [ ] 教建模手怎么调包（不要太深入，够用就行）
- [ ] 和写作手配合：确保代码输出图能直接放进论文
- [ ] 建立 `templates/` 目录，整理所有代码模板

### 代码模板建议结构
```
templates/
├── data_processing/
│   ├── read_data.py          # 读各种格式
│   ├── clean_data.py         # 缺失值+异常值
│   └── feature_engineering.py # PCA/标准化
├── models/
│   ├── regression.py          # 线性回归+SVR
│   ├── classification.py      # 决策树+SVM+随机森林
│   ├── clustering.py          # K-means+DBSCAN
│   ├── optimization.py        # 线性规划+整数规划
│   └── timeseries.py          # ARIMA
├── visualization/
│   ├── basic_plots.py         # Matplotlib模板
│   └── interactive_plots.py   # Plotly模板
└── utils/
    ├── metrics.py             # 评价指标
    └── report.py              # 自动生成结果表格
```

### 第5周（交叉补位）

- [ ] 和编程手确定图表格式统一规范

### 论文写作核心能力
```
Day1 开始写 → 引言+问题重述（框架先搭好，后期填充）
Day2 建模手确定模型后 → 模型假设+模型建立（写框架+公式）
Day3 上午 → 结果分析+模型检验
Day3 下午 → 摘要！（反复改，三人一起审）
Day3 晚上 → 排版+参考文献+最终检查
```

### 关键：摘要写作
参考 [[论文写作指南]] 中的摘要公式。
