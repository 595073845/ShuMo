# 论文写作指南

> **重要排名**: 摘要(50%) > 模型(20%) > 结果(15%) > 排版(10%) > 其他(5%)  
> **所有三个人都要读这篇**，不能只让写作手一个人看。

---

## 论文标准结构（CUMCM格式）

```
1. 摘要       ← 最重要，单独一页
2. 关键词     ← 3-5个
3. 问题重述   ← 1-2页，不可照抄题目
4. 问题分析   ← 1-2页，展示思路过程
5. 模型假设   ← 半页，假设要合理且有依据
6. 符号说明   ← 半页，表格形式
7. 模型建立与求解 ← 核心，5-10页
   7.1 模型一 ...
   7.2 模型二 ...
8. 模型检验   ← 1-2页，敏感性分析/误差分析
9. 模型评价   ← 1页，优缺点
10. 参考文献  ← 半页
11. 附录      ← 核心代码
```

## 摘要——论文的灵魂

### 摘要的重要性
- 阅卷老师先看摘要，决定给一等、二等还是三等
- **摘要不过关直接二等以下**
- 字数严格限制：**400-500字**

### 摘要的万能公式

```
第一段（背景+问题）：1-2句话
  "针对XXX问题，本文建立了XXX模型，采用了XXX方法，解决了XXX。"

第二段（方法+模型）：3-5句话
  "对于问题一，建立了XXX模型，使用XXX算法求解，得到XXX结果。
   对于问题二，..."
  （每个问题一句）

第三段（结果+结论）：1-2句话  
  "结果表明，XXX。模型的优势在于XXX。"
```

### 摘要写作原则
- ✅ 每句话都要有信息量（不写废话）
- ✅ 使用具体数字和结果（"准确率达到95.2%"而不是"准确率较高"）
- ✅ 每个问题至少对应一个模型
- ❌ 不要出现"本文"/"我们"过多的自我指代
- ❌ 不要写背景知识（阅卷老师都知道）

### 摘要修改流程
```
初稿 → 三人各自改一版 → 合并 → 再次润色 → 定稿
至少改5轮，不要第一次写完就不动了。
```

## 模型假设怎么写

### 正确示范
```
假设1：数据不存在系统误差，测量误差服从正态分布。
假设2：短期内各影响因素之间不存在交互作用。
假设3：原材料供应充足，不考虑供应链中断风险。
```

### 原则
- 假设要**合理** → 说明为什么这样假设
- 假设要**有用** → 每个假设后面简化了哪个计算
- 不要写废话假设（"假设地球是球体"——如果和问题无关）

## 公式与排版要求

### LaTeX推荐配置
```latex
% 公式示例
\begin{equation}
    y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \varepsilon
\end{equation}
```

### 排版原则
- 图要有图题（图1 xxx），表要有表题（表1 xxx）
- 公式要编号，正文中引用时用编号
- 图表清晰，线条粗细一致，颜色方案一致
- **全篇文章风格统一**（字体、字号、行距）

### 必须避免
- ❌ 截图放到论文里（公式要打出来，不要截图）
- ❌ 表格跨页（用长表格或调整）
- ❌ 参考文献格式不统一
- ❌ 页眉页脚乱码

## 使用AI辅助写作

### ✅ 可以
- 让AI润色段落（给AI一段你自己写的文字，要求"保持原意，提升学术性"）
- 让AI对摘要提修改意见
- 让AI校对英文摘要
- 让AI生成LaTeX表格模板
- 编译报错贴给AI

### ❌ 不可以
- AI直接生成论文正文（查重100%过不去）
- AI生成的数据和结论（必须基于你的模型实际结果）
- 完全照搬AI的表述（要改写成自己的语言）

详见 [[AI使用红线]]。

## 参考文献格式

### GB/T 7714格式（中文常用）
```
[1] 姜启源, 谢金星, 叶俊. 数学模型[M]. 北京: 高等教育出版社, 2015.
[2] 司守奎, 孙兆亮. 数学建模算法与应用[M]. 北京: 国防工业出版社, 2020.
```

### 外文文献
```
Smith J, Johnson K. A survey of optimization methods[J]. Journal of Optimization, 2022, 15(3): 234-256.
```

## 附录（代码）
- 只放核心代码，不要全部粘贴
- 代码要有注释
- 确保代码可读性（比赛后可能会有代码抽查）

## 最后检查清单

提交前逐项检查：
- [ ] 摘要格式正确、字数不超、每个问题都提到了
- [ ] 所有公式正确、编号连续
- [ ] 图表有标题、编号、引用
- [ ] 参考文献格式统一
- [ ] 页眉页脚正确
- [ ] PDF转为论文格式没问题
- [ ] 三个人各读一遍（找错别字和逻辑漏洞）
