# AI 工具清单

> **定位**: AI是生产力工具，不是作弊器  
> **原则**: 做加法不做减法 — 人脑做关键决策，AI加速执行  
> **准备时间**: 赛前3天确认所有工具可用

## 推荐组合方案

### 免费方案（完全够用）
```
ChatGPT（免费版）+ GitHub Copilot（学生免费）+ Kimi + DeepL
```
- ChatGPT: 概念解释、debug、头脑风暴
- Copilot: 代码补全
- Kimi: 中文润色
- DeepL: 英文翻译

### 进阶方案（推荐）
```
ChatGPT Plus（$20/月）+ GitHub Copilot + Cursor + DeepL
```
- 比免费版多：Code Interpreter（直接上传数据做分析）
- 比赛期间一个月Plus足够了

## AI使用技巧

### 问概念
```
提示词："用大一的水平解释X，举一个竞赛中可能遇到的例子"
```

### 问模型选择
```
提示词："我要解决X问题，数据有N条、M个特征、输出是Y，
给出3个可用的模型，从精度、计算复杂度、数据需求三方面对比"
```

### 代码debug
```
提示词："这段代码报错X，帮我找出问题并修复。
代码：<粘贴代码>，错误信息：<粘贴报错>"
```

### 论文润色
```
提示词："帮我润色这段摘要，保持学术风格，
原意不变，提升表达的精确性和流畅性：
<粘贴自己的文字>"
```
