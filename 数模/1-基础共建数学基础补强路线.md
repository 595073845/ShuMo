# 数学基础补强路线

> **目标**: 从"学过"到"会用"  
> **原则**: 不求深证，只求会用、会选、会解释  
> **时间**: 第1周（7.14 - 7.20）

---

## 一、线性代数（矩阵是数模的通用语言）

### 必须掌握

| 知识点 | 用途 | 对应模型 |
|--------|------|----------|
| 矩阵乘法/转置/逆 | 基础运算，一切模型的基础 | 所有模型 |
| 特征值与特征向量 | 降维、主成分分析的核心 | PCA、谱聚类 |
| 奇异值分解(SVD) | 数据压缩、推荐系统 | 图像压缩、降维 |
| 最小二乘法 | 拟合、回归问题的数学本质 | 线性回归 |
| 正定矩阵 | 判断目标函数凹凸性 | 优化问题 |

### 怎么学
- 复习教材重点，不要从头看到尾
- 重点做**矩阵分解**相关题目（SVD、特征分解）
- [[推荐网课]] 中 3Blue1Brown 的线性代数系列（直觉理解）
- 用 NumPy 实操：`np.linalg.svd()`, `np.linalg.eig()`

### 不用学的
- Jordan标准形、矩阵函数的深入推导
- 行列式展开的巧妙技巧（计算器会算）

## 二、概率论与统计（处理不确定性的工具）

### 必须掌握

| 知识点 | 用途 | 对应模型 |
|--------|------|----------|
| 贝叶斯公式 | 逆向概率、分类模型的核心 | 贝叶斯分类、贝叶斯优化 |
| 假设检验（t检验、F检验） | 判断两组数据是否有显著差异 | 数据分析 |
| 回归分析 | 找变量关系 | 线性回归、多元回归 |
| 蒙特卡洛模拟 | 随机模拟、求复杂积分 | 随机模拟 |
| 正态分布/大数定律/中心极限定理 | 统计推断的基础 | 几乎所有统计方法 |
| 方差分析(ANOVA) | 多组间差异检验 | 实验数据分析 |

### 怎么学
- 重点理解 **假设检验的 p 值含义**（比赛中常用）
- 至少手算一次回归分析的推导（理解最小二乘）
- Python：`scipy.stats` 包里实现各种统计检验

## 三、优化理论（数模的核心引擎）

### 必须掌握

| 知识点 | 用途 | 对应模型 |
|--------|------|----------|
| 线性规划(LP) | 资源分配、生产调度 | 运输问题、投资组合 |
| 整数规划(IP) | 离散决策变量 | 指派问题、选址问题 |
| 非线性规划(NLP) | 更复杂的约束优化 | 各种实际优化问题 |
| 多目标优化 | 多个目标同时优化 | Pareto最优解 |
| 梯度下降思想 | 优化算法底层原理 | 神经网络训练 |

### 怎么学
- 会用 **Python 调包**（`scipy.optimize`）解优化问题即可
- 不需要手动实现单纯形法，但要理解思路
- 重点看 **约束条件怎么转换为数学表达式**
- 比赛中最常用的就是线性规划 + 整数规划

### 经典例题
```
某工厂生产 A、B 两种产品：
- A 利润 40元/件，B 利润 30元/件
- 原料限制：A需原料2kg/件，B需原料1kg/件，共120kg
- 工时限制：A需1h/件，B需2h/件，共160h
- 问各生产多少件利润最大？
→ 决策变量 x₁,x₂ → 目标函数 max 40x₁+30x₂ → 约束条件 ...
```

## 四、微分方程（动态系统建模）

### 了解即可
- 一阶/二阶常微分方程(ODE)的求解思想
- 差分方程 → 离散化建模
- 人口模型（Logistic）、传染病模型（SIR）
- **知道什么情况用微分方程**，求解交给计算机

### Python实现
```python
from scipy.integrate import odeint
import numpy as np

def model(y, t):
    dydt = -0.1 * y  # 示例：指数衰减
    return dydt

y0 = 5  # 初始值
t = np.linspace(0, 20, 100)  # 时间点
y = odeint(model, y0, t)
```
