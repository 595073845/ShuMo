# 模拟赛 SOP

> **时间**: 第6周（8.18 - 8.24）  
> **模拟赛#1**: 禁用AI（8.18-8.20）  
> **模拟赛#2**: AI全开（8.22-8.24）  
> **前提**: 前5周的计划已按[[备赛时间线]]执行

---

## 模拟赛目的

- 暴露真实短板（而不是"我以为我会了"）
- 磨合三人协作节奏
- 测试代码模板和论文模板的可用性
- 建立比赛72小时的时间感

---

## 模拟赛#1：禁用AI

### 目标
测真实水平，暴露短板。这次可以做得不好，但要真实。

### 题目选择
- 建议选择 **2022或2023年的C题**（数据类，适合IC背景）
- 提前下载好题目和数据，不要提前看答案

### 时间安排
```
Day1 (8.18)
  08:00 发题 → 各自读题30min → 讨论1h → 确定选题
  10:00-12:00 问题分析+模型初步设计
  14:00-18:00 模型建立+初步编程
  20:00-22:00 继续推进，写作手开始写引言+问题重述

Day2 (8.19)
  08:00-12:00 建模求解
  14:00-18:00 继续求解+调参
  20:00-22:00 初步结果整理

Day3 (8.20)
  08:00-12:00 结果分析+补图
  14:00-18:00 论文撰写冲刺
  18:00-20:00 摘要（重中之重！）
  20:00-22:00 排版+检查+提交
```

### 复盘（8.21，模拟赛#2之前）

三人坐在一起，逐项回答：

#### 选择题
- [ ] 选题花了多久？有没有犹豫不决？
- [ ] 选题后后悔了吗？为什么？

#### 建模题
- [ ] 模型选得快吗？还是纠结了很久？
- [ ] 有没有模型建好了但实现不出来的情况？
- [ ] 模型假设写得合理吗？

#### 编程题
- [ ] Python环境有没有出问题？
- [ ] 数据清洗花了多久？和预期一致吗？
- [ ] 代码有没有跑不出结果的情况？

#### 写作题
- [ ] 什么时候开始写的？太晚了还是合理？
- [ ] 摘要写了几版？满意吗？
- [ ] 排版花了多少时间？有没有浪费时间的地方？

#### 团队协作题
- [ ] 三人沟通是否顺畅？
- [ ] 有没有"等别人"的时间浪费？
- [ ] 作息安排合理吗？

## 模拟赛#2：AI全开

### 目标
练"人机协作节奏"，找到最优的AI使用方式

### AI使用策略
- 用 [[竞赛72h-AI节奏SOP]] 作为指导
- 每次用AI后记录：用了什么、效果如何、花了多少时间
- 尝试不同的AI分工方式

### 复盘（8.25之前）

- [ ] AI带来了多少效率提升？量化（如"本来要2h，AI辅助1h完成"）
- [ ] 哪些环节AI最有帮助？哪些环节AI帮倒忙？
- [ ] 有没有过度依赖AI导致的问题？
- [ ] 修正 [[Prompt模板库]]（把这次好用的prompt加进去）
